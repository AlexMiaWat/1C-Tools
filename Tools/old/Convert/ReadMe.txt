1С:Предприятие 8
Конфигурация "Конвертация данных", редакция 2.1
Версия 2.1.8.2
=================================================================

В поставку включены внешние обработки для работы с конфигурациями на 
платформах "1С:Предприятие 7.7", "1С:Предприятие 8". Все обработки расположены 
в подкаталоге tmplts\1c\Conversion\2_1_8_2\ каталога установки 1С:Предприятия 8.

1. "Универсальный обмен данными в формате XML" (V8Exchan82.epf) - предназначена
для выгрузки данных в файл из любой конфигурации, реализованной на
платформе "1С:Предприятие 8.2" и последующей загрузки данных из файла
в любую конфигурацию.
2. "Универсальный обмен данными в формате XML" (V8Exchan83.epf) - предназначена
для выгрузки данных в файл из любой конфигурации, реализованной на
платформе "1С:Предприятие 8.3" и последующей загрузки данных из файла
в любую конфигурацию.
3. "Универсальная выгрузка данных в формате XML" (V77Exp.ert) - предназначена
для выгрузки данных в файл из любой конфигурации, реализованной на
платформе "1С:Предприятие 7.7".
4. "Универсальная загрузка данных в формате XML" (V77Imp.ert) - предназначена
для загрузки данных в файл в любую конфигурацию, реализованную на
платформе "1С:Предприятие 7.7".
5. "Выгрузка описания структуры метаданных конфигурации 8.2" (MD82Exp.epf) - предназначена
для выгрузки описания структуры метаданных любой конфигурации, реализованной на
платформе "1C:Предприятие 8.2".
6. "Выгрузка описания структуры метаданных конфигурации 8.3" (MD83Exp.epf) - предназначена
для выгрузки описания структуры метаданных любой конфигурации, реализованной на
платформе "1C:Предприятие 8.3".
7. "Выгрузка описания структуры метаданных конфигурации 7.7" (MD77.ert) - предназначена
для выгрузки описания структуры метаданных любой конфигурации, реализованной на
платформе "1С:Предприятие 7.7".
8."Регистрация изменений для обмена" (РегистрацияИзмененийДляОбмена82 (обычное приложение)) - предназначена
для просмотра объектов, для которых есть регистрация изменений, реализованной на
платформе "1С:Предприятие 8.2" только в  режиме толстого клиента (обычное приложение).
9."Регистрация изменений для обмена" (РегистрацияИзмененийДляОбмена82 (управляемое приложение)) - предназначена
для просмотра объектов, для которых есть регистрация изменений, реализованной на
платформе "1С:Предприятие 8.2".
10."Регистрация изменений для обмена" (РегистрацияИзмененийДляОбмена83 (управляемое приложение)) - предназначена
для просмотра объектов, для которых есть регистрация изменений, реализованной на
платформе "1С:Предприятие 8.3".
11. "Выгрузка загрузка данных XML" (ВыгрузкаЗагрузкаДанныхXML82.epf)  - предназначена для обмена данными между конфигурациями
одинаковыми по метаданным, реализованными на платформе "1С:Предприятие 8.2" только в  режиме толстого клиента (обычное приложение).
12. "Выгрузка загрузка данных XML" (ВыгрузкаЗагрузкаДанныхXML83.epf)  - предназначена для обмена данными между конфигурациями
одинаковыми по метаданным, реализованными на платформе "1С:Предприятие 8.3".
13. Две информационные базы, на которых построены примеры типичного решения часто возникающих проблем.
Информационная база источник (Source.dt) и информационная база приемник (Dest.dt).
